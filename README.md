## 数据来源
- 股票行情：akshare，后复权，日度
- 市场指数：akshare，沪深300(000300)、中证500(000905)
- 宏观指标：CPI同比增速(akshare)、M2同比增速(akshare)
- 财务数据：akshare/baostock

## 存储方式
- 基础：CSV（方式A）
- 进阶：Parquet（方式B）- 选择理由：列式存储适合大数据量分析，压缩率高

## 如何运行
1. 安装依赖：`pip install -r requirements.txt`
2. 运行 `01_download.ipynb` 下载原始数据
3. 运行 `02_clean.ipynb` 清洗并存储数据
4. 运行 `03_analysis.ipynb` 查看分析结果
5. 打开 `report.html` 阅读完整报告
